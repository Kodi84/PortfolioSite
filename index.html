Hello, my name is Cung Tran. I'm an ambitious Web Developer live in Orange County, CA. I have experience building websites and applications using both front and back-end technologies. If you have a project or creative need that I can help with, please get in touch.

About Me

As you're reading this, I am following my dream and my passion as a web developer. After graduating from a Web Development School, I've been actively create applications, signed up for every supplemental learning site out and continue to teach myself new technologies.

I enjoy being challenged and working on projects that require me to work outside my comfort and knowledge set. I have not only learned new programing languages but also developed techniques which are invaluable and make me a better employee.

I love helping people with marketing, web design, web development and marketing strategy. I also started a blog, cungblog, where I share inspiration, tutorials, and resources.

When not coding I'm most likely playing badminton, soccer, or catching up on Walking Dead. I am a lover of science and all things tech! I cook Pho and make Papaya salad at least once a month.

Skill

What I can do

  • Create website layout/user interface by using front end technology and integrate data from back-end services and databases
  • Knowledge of relational database systems, Object Oriented Programming and how MVC frameworks operate
  • Experience with Git control and Github to keep track of projects.
  • Adapt quickly on learning the modern technologies
  • Highly motivated and self-starter that is able to work independently or as a member of a team
